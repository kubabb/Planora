# Planora
https://planora-web-beryl.vercel.app/
> Project planning, mind maps, and architecture diagrams generated by AI and rendered locally.

Planora turns project ideas into structured Markdown plans, interactive mind maps, and Mermaid architecture diagrams. It is designed to work as a standalone product through a CLI, VS Code extension, and local web app.

## Product vs Dev Environment

Planora itself should run without Hermes.

- Product runtime: Planora uses its own agent and an API key for a model provider such as OpenRouter, OpenAI, or Ollama.
- Dev environment: this repository may still include Hermes-oriented files such as `SOUL.md`, `AGENTS.md`, MCP notes, and workflow conventions. Those exist to help build Planora, not as a requirement for Planora users.

## What It Does

```bash
planora init
planora plan --ai
planora web
```

### Generated Files

| File | Purpose |
|------|---------|
| `PROJECT_PLAN.md` | Overview, MVP, tech stack, milestones |
| `ROADMAP.md` | Phased development timeline |
| `MINDMAP.md` | Hierarchical outline -> interactive mind map |
| `ARCHITECTURE.md` | System, data flow, and component diagrams |
| `AGENT_SETUP.md` | Planora agent config, provider, model, workflow notes |
| `planora.json` | Project metadata |

### Visualizations

- Mind Map: `MINDMAP.md` rendered as an interactive mind map
- Architecture Graphs: `ARCHITECTURE.md` rendered as Mermaid diagrams
- Dashboard: all projects in one view

## Quick Start

```bash
# install (coming soon)
npm install -g planora

# configure Planora's own agent
planora config

# create a new project
planora init --name "my-app" --stack "react,node,postgres"

# generate planning docs with AI
planora plan --ai

# open the local dashboard
planora web
```

OpenRouter is the recommended default provider for first-run setup.

Hermes is not required for this flow.

## Monorepo Structure

```text
packages/
|- core/        # @planora/core - models, generators, storage, AI client
|- cli/         # @planora/cli - terminal commands
|- vscode-ext/  # @planora/vscode-ext - VS Code integration
|- web/         # @planora/web - React + Vite dashboard
`- runner/      # @planora/runner - Planora agent runtime, Hermes bridge optional
```

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Language | TypeScript (strict) |
| Runtime | Node.js 20+ |
| Monorepo | npm workspaces |
| CLI | Commander.js |
| Web | React 18 + Vite + Tailwind |
| Diagrams | Mermaid.js |
| Mind Maps | markmap.js |
| Database | SQLite (local-first), optional Supabase |
| AI Agent | Planora own agent via OpenAI-compatible APIs |
